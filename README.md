# RdpSession
Automatic Log in to your Server via Remote Desktop Protocol (RDP) with PowerShell

.Synopsis
    Connect RDP from PowerShell
    
.DESCRIPTION
    Automatic RDP Connection from PowerShell
    
.EXAMPLE
    New-RdpSession -Computer ts.contoso.com
